# It-s-ok-not-to-be-ok

# 🌿It's ok not to be ok Mental Wellness Web App

A responsive and interactive **Mental Health Support Platform** designed to help users relax, express emotions, and access wellness resources in a simple and engaging way.

---

## 💡 Project Overview

This project focuses on promoting **mental well-being** through various features like counseling support, creative therapy, mindfulness activities, and interactive experiences.

It provides a safe and calming digital space where users can:

* Explore wellness resources
* Express themselves through art
* Access counseling support
* Relax with engaging activities

---
